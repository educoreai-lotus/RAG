How to Embed the RAG Chat Widget

Option 1: Script Tag

<script src="https://your-domain.com/widget.js"></script>
<script>
  RAGWidget.init({
    apiUrl: 'https://api.your-domain.com',
    tenantId: 'your-tenant-id'
  });
</script>

Option 2: React Component

import FloatingChatWidget from '@rag-microservice/widget';

function App() {
  return <FloatingChatWidget apiUrl="https://api.your-domain.com" />;
}

Option 3: iframe

<iframe 
  src="https://widget.your-domain.com"
  width="400"
  height="600"
  frameborder="0"
></iframe>
